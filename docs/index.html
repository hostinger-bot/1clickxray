Overview

System Requirements

This script is designed for:

  • Ubuntu operating system
  • Debian operating system

Important Notes

  • After installation, type menu in terminal to display available options
  • This script is safe for reinstallation without requiring VPS rebuild
  • Supports IPv6-Only VPS with specific domain setup requirements

IPv6 Only VPS Setup Notice

When using IPv6 Only VPS, during domain setup you must choose option #2 (use your own domain). Option #1 is not supported for IPv6 Only configurations.
